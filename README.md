# Fire Extinguisher Game
Play as a fire extinguisher defeating an evil DJ, made with Raylib! It supports Mac and Windows.

## Downloading
1. Download the repo.
2. Run the appropriate script in the Scripts/ directory on Windows, and DO NOT RUN THE MACOS SCRIPT WHEN USING MAC.
3. On mac, run Vendor/Binaries/Premake/macOS/premake5 --file=Build.lua xcode4 in the root directory in Terminal
